Detective Quest: Nível Mestre 
🔍Este projeto é um simulador de investigação técnica desenvolvido em C++, focado na integração de múltiplas estruturas de dados para resolver um mistério lógico.
 O jogador explora uma mansão, coleta evidências e utiliza um sistema de julgamento automatizado para determinar a culpa de um suspeito.
 🚀 Objetivo do Projeto:O sistema foi projetado para demonstrar o uso prático de Árvores Binárias, Árvores de Busca (BST) e Tabelas Hash em um cenário de jogo interativo.
 🛠️ Estruturas de Dados AplicadasÁrvore Binária (Mansão): Utilizada para representar a estrutura física e a navegação da mansão, onde cada nó é um cômodo com caminhos para esquerda ou direita.
 Árvore de Busca Binária - BST (Inventário): Armazena as pistas coletadas de forma ordenada, permitindo uma gestão eficiente do inventário de evidências.
 Tabela Hash (Lógica de Suspeitos): Realiza a associação rápida entre uma pista (chave) e o respectivo suspeito (valor), garantindo performance na fase de acusação.
 Structs e Ponteiros: Implementação de alocação dinâmica para gerenciar os dados compostos de salas e pistas.
 🎮 FuncionalidadesExploração Interativa: Navegação pelos cômodos da mansão através de comandos simples (e para esquerda, d para direita).
 Coleta de Evidências: Identificação automática de pistas ao entrar em salas específicas.
 Sistema de Julgamento: Verificação lógica que exige pelo menos duas pistas válidas para sustentar uma acusação contra um suspeito.Processamento Recursivo: Uso de recursividade para explorar a mansão e contabilizar as provas durante o veredito final.
 📋 Requisitos Técnicos Atendidos: 
 Modularidade: Funções específicas para criação de salas, inserção de pistas e consulta na tabela hash.
 Documentação Interna: Comentários detalhados em funções críticas como criarSala(), explorarSalas() e verificarSuspeitoFinal().
 Mapa Estático: Configuração manual da mansão no main() para garantir a integridade do cenário de teste.
